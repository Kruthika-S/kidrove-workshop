I made the AI and Robotics Summer Workshop website using React.js and Tailwind CSS. I wanted to make it easy to update and keep organized. I used a component-based architecture. The website has a lot of features including a hero section that looks good on any device details about the workshop what you can learn, a section with asked questions that you can click on to get the answers and a form to sign up.

The backend of the website was made using Express.js. It has an endpoint that checks what the user types in and then handles the sign up requests. I also added some things to make the website nicer to use like checking to make sure the form is filled out right showing a message when something is loading and a success message when you sign up. The website also scrolls smoothly to the sign up form when you click the Enroll button. You can see the website on Netlify, which makes it easy to look at and test.
